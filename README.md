###Instalation

1. Python 2.7 - required
2. Google Chrome installed on your system
3. Create a virtual environment `python -m venv venv`
4. Activate venv Ubuntu `source venv/bin/activate`, Windows `\env\Scripts\activate.bat`
5. Install dependencies `pip install -r requirements.txt`

### Run script
1. Activate venv
2. run `python app.py --username <your instagram_username> --password <your_instagram_password> --file
 <full_path_to_your_txt_file>`
3. Review your post on Instagram
